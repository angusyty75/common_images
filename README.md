# Common Images Repository

This repository contains common image assets used across projects.

## Repository Contents

This repository includes the following files:

- **card_icons.png** - Collection of card icons
- **logo.jpg** - Logo in JPG format
- **logo.png** - Logo in PNG format

## How to View Repository Files

### Method 1: View Files on GitHub

You can view the files directly on GitHub:

1. Navigate to the repository: https://github.com/angusyty75/common_images
2. Click on any file to view it in your browser
3. For images, GitHub will display a preview automatically

### Method 2: Clone the Repository

To download all files to your local machine:

```bash
git clone https://github.com/angusyty75/common_images.git
cd common_images
```

After cloning, you can open the image files with any image viewer.

### Method 3: Download Individual Files

To download a specific file:

1. Navigate to the file on GitHub
2. Click on the file name (e.g., `logo.png`)
3. Click the "Download" button or right-click on the image and select "Save image as..."

### Method 4: Use Raw File URLs

You can access files directly via their raw URLs:

- card_icons.png: `https://raw.githubusercontent.com/angusyty75/common_images/main/card_icons.png`
- logo.jpg: `https://raw.githubusercontent.com/angusyty75/common_images/main/logo.jpg`
- logo.png: `https://raw.githubusercontent.com/angusyty75/common_images/main/logo.png`

## Usage in Projects

You can reference these images in your projects using the raw URLs above, or download them locally and include them in your project's assets directory.
